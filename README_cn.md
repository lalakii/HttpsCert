# Https 证书生成器 for Windows

[ [English](./README.md) | [简体中文](./README_cn.md) ]

为了更加方便地生成SSL证书用于测试，我制作了这个小工具

密码字段是可选项，当你需要生成.pfx格式的证书文件时，才需要设置它

基于 [BouncyCastle](https://github.com/bcgit/bc-csharp) 实现
